# ELARA — Yeshee Agarwal’s Digital Twin

ELARA is an AI-powered personal portfolio chatbot built using Generative AI and Retrieval-Augmented Generation (RAG). It allows visitors to interact with an intelligent assistant trained on Yeshee Agarwal’s resume, projects, and experience.

Instead of reading a static portfolio, recruiters can ask ELARA questions about Yeshee’s work, skills, and achievements.

## Features
- AI chatbot trained on personal documents
- Retrieval-Augmented Generation for accurate responses
- Conversation memory for follow-up questions
- Modern dark UI with a purple theme
- Built with Groq LLaMA models

## Tech Stack
Python, Streamlit, LangChain, FAISS, HuggingFace Embeddings, Groq LLaMA

## How to Run
1. Install dependencies  
   pip install -r requirements.txt  

2. Add your Groq API key  
   Set GROQ_API_KEY in your environment or Streamlit secrets  

3. Run the app  
   streamlit run app.py
